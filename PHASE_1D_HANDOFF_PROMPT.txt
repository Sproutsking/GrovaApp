**UI Sections**:

1. **Wallet Selection** (Grid of detected wallets):
   - Show all wallets from `useWalletConnect` hook
   - OnClick: Connect wallet, set phase to "connecting"
   - Show connected state with checkmark

2. **Chain Selector** (Dropdown):
   - Polygon (default)
   - Ethereum
   - Base
   - Arbitrum
   - Optimism
   - Solana
   - Tron
   - Cardano
   - Updates gas fee display on change

3. **Amount Input** (Number field):
   - Min: $1
   - Max: $10,000
   - Show USD symbol
   - On input, calculate gas fee from chain

4. **Gas Fee Display**:
   - Format: "Est. Gas: $XX.XX"
   - Pull from `web3PaymentService` constants (SUPPORTED_CHAINS)
   - Update when chain changes

5. **Phase-Based UI**:
   - `null`: Show "Pay Now" button (enabled if wallet connected & amount filled)
   - `"connecting"`: Show "Connecting..." (disabled)
   - `"signing"`: Show "Sign in wallet..." (disabled)
   - `"confirming"`: Show **Progress Bar** + confirmation counter
   - `"completed"`: Show "✓ Done!" + success message

6. **Confirmation Progress Bar** (During confirming phase):
   ```
   [████████░░░░░░] 8/12 Confirmations
   Waiting for confirmations (8/12)
   ```
   - Fill width: `(currentConfirmations / requiredConfirmations) * 100%`
   - Update real-time via `pollPaymentStatus` callback

7. **Error Display**:
   - Show `signatureError` if signature rejected
   - Show "Transaction failed" if payment fails
   - Button to retry

## 🔧 Implementation Steps

### Step 1: Use Existing Services
```javascript

```

These already have:
- `useWalletConnect()` returns `{wallets, connected, connecting, connectedAddress, connectWallet}`
- `web3PaymentService.initiateAutoPayment()` returns `{sessionId, nonce, treasuryAddress, amountFormatted, estimatedGas}`
- `web3PaymentService.submitAutoPayment()` returns `{paymentId, txHash, requiredConfirmations}`
- `web3PaymentService.pollPaymentStatus()` takes `{paymentId, onStatusChange, onConfirmed, onFailed, maxWaitMs}`

**Do not modify these services.**

### Step 2: Flow Handler
```javascript
const handleQuickPaySubmit = async () => {
  // 1. Initiate (get nonce + treasury address)
  // 2. Request signature from wallet
  // 3. Submit TX to backend
  // 4. Poll confirmations in real-time
  // 5. Auto-credit on finalization
};
```

### Step 3: Real-Time Confirmation
```javascript
await web3PaymentService.pollPaymentStatus({
  paymentId,
  onStatusChange: (status) => console.log(status),
  onConfirmed: (confirmations) => {
    setCurrentConfirmations(confirmations);
    setQuickPayPhase("completed");
    if (onRefresh) onRefresh();
    // Optional: play success sound
  },
  onFailed: (error) => {
    setQuickPayPhase(null);
    setSignatureError(error.message);
  },
  maxWaitMs: 600_000 // 10 min timeout
});
```

### Step 4: Export & Use in DepositTab
**In DepositTab.jsx**, add import and render:
```javascript
import QuickPayMode from "./QuickPayMode";

// In main component return, add tab buttons:
<button onClick={() => setMode("quick")}>Quick Pay</button>

// Render based on mode:
{mode === "quick" && <QuickPayMode {...props} />}
```
